Getting Started

One import, full system

Import the token file once in your HTML or CSS. All CSS custom properties and utility classes become available globally.

HTML 
<link rel="stylesheet" href="design-tokens.css" />

<!-- Google Fonts (Inter + JetBrains Mono) -->
<link href="https://fonts.googleapis.com/css2?family=Inter:wght@400;500;600;700&family=JetBrains+Mono:wght@400;500&display=swap" rel="stylesheet" />
Use tokens in your CSS
CSS 
/* Use any CSS custom property directly */
.my-card {
  background: var(--color-bg-card);
  border: 1px solid var(--color-border);
  border-radius: var(--radius-lg);
  padding: var(--space-6);
  box-shadow: var(--shadow-sm);
}

.my-heading {
  font-size: var(--font-size-xl);
  font-weight: var(--font-weight-semibold);
  color: var(--color-text-primary);
}

Accent Card

<div class="card">
  <h3 class="text-heading-md">Title</h3>
  <p  class="text-body-sm text-secondary">Description</p>
</div>

<!-- CSS (in design-tokens.css) -->
.card {
  background: var(--color-bg-card);
  border: 1px solid var(--color-border);
  border-radius: var(--radius-lg);
  box-shadow: var(--shadow-sm);
  padding: var(--card-padding); /* 24px */
}
