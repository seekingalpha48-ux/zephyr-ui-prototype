Navigation Spec

Bottom tab bar height
82px total (66px bar + 16px iOS safe-area bottom inset). Android: 56px + system gesture inset.
Tab count
5 tabs: Home · Contacts · Invoices · Pipeline · More (⋯ opens drawer). Mirrors desktop sidebar top-5 nav items.
Active indicator
4px indigo pip below active tab icon. Label color changes to --color-primary-600. Icon stays emoji (swap to Lucide on native).
Badge (notification dot)
16px danger-500 circle, white text, 2px white border, absolute positioned top-right of icon.
Off-canvas drawer width
280px, slides in from left with translateX(-100%) → 0. Overlay: rgba(0,0,0,0.45). Close on overlay tap or swipe-left.
Drawer animation
0.28s cubic-bezier(0.4, 0, 0.2, 1) — matches Material Motion standard easing.
FAB
52px circle, primary-600 fill, shadow-lg, positioned bottom: 96px; right: 16px (above tab bar). Opens context-aware bottom sheet.
Bottom sheet
Slides up from bottom (0.3s cubic-bezier). Handle bar = 36×4px neutral-300 pill. Dismiss on overlay tap or drag-down.
Breakpoint
Bottom nav shown at < 768px. Desktop sidebar shown at ≥ 1280px. Tablet (768–1279): collapsible icon-only sidebar.
